# haha-edu
an edu-h5-demo based on vue&amp;vant
<p><a href="https://hanxyan.github.io/haha-edu/#/index">预览地址</a></p>

## 说明    
**基础** ：vue2 + vuex + vue-router + webpack + ES6/7 + sass + flex + svg   
**UI框架** ：vant       
**图标** ：iconfont  



## 完成内容   
基本的tab静态页面

## 项目运行
### 安装依赖
```
npm install
```

### 开发环境运行
```
npm run serve
```

### 打包
```
npm run build
```

### 测试
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
